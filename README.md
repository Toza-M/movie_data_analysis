# movie_data_analysis

## Project Overview
This project focuses on analyzing movie datasets to extract meaningful insights. It involves data integration, cleaning, and visualization to understand trends and patterns in the movie industry.

## Folder Structure
- `data/`: Contains raw and processed datasets.
  - `imdb_data.csv`: Dataset from IMDb.
  - `movies_dataset_final.csv`: Final integrated dataset.
  - `movies_financial.csv`: Financial data of movies.
  - `rotten_tomatoes_movies.csv`: Dataset from Rotten Tomatoes.
- `data_integration.ipynb`: Notebook for integrating and cleaning datasets.
- `web_scrapping.ipynb`: Notebook for web scraping movie data.
- `questions (1).ipynb` and `Questions (2).ipynb`: Notebooks for answering specific analytical questions.
- `README.md`: Project documentation.

## Getting Started
1. Clone the repository.
2. Install the required Python libraries.
3. Open the Jupyter notebooks to explore the analysis.

## Requirements
- Python 3.x
- Libraries: pandas, numpy, matplotlib, seaborn, etc.
